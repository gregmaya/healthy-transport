Healthy Transport
Score mode

Baseline — evaluates the geometric quality of a location using the working-age walk curve. No population data. Equal weight per network node. Useful for assessing long-term infrastructure regardless of who lives there today.

Contextual — weights each location by the actual age-disaggregated population living nearby (modelled estimates). Scores reflect today's residents. Low/High show the uncertainty range of the demographic model.

Each group uses a different benefit curve — the optimal walk distance shifts with age and mobility. Scores reflect how much health benefit a stop at that location would generate for people in that group.
